# 記憶語言天賦測驗 · Memory Language Talent

一個獨立、零依賴的單檔測驗（`index.html`），靈感參考自
[memolingua](https://jjfishjj.github.io/memolingua/) 的方向，但改用
**8 種記憶天份 × 5 大學習指標** 的架構，並輸出完整個人化報告。

## 內容架構

**8 種記憶天份類型**
| 類型 | 記憶天份 | 適合策略 |
|---|---|---|
| 🎨 圖像建構者 | 靠畫面、空間、顏色記憶 | mind map、圖像單字卡、場景記憶 |
| 🎵 聲音模仿者 | 靠語音、節奏、語調記憶 | shadowing、朗讀、音檔循環 |
| 📝 文字整理者 | 靠筆記、定義、分類記憶 | 摘要、表格、語法規則、錯題本 |
| 🎭 情境行動者 | 靠操作、角色扮演、任務記憶 | 對話演練、任務式學習、沉浸場景 |
| 💬 社交輸出者 | 靠分享、教人、互動記憶 | 小組討論、教學輸出、口說陪跑 |
| 📊 系統累積者 | 靠規律、數據、複習節奏記憶 | Anki、streak、間隔複習、進度追蹤 |
| 🧩 創意連結者 | 靠比喻、故事、聯想記憶 | 記憶宮殿、故事化單字、跨域連結 |
| 💼 商業應用者 | 靠目標、成果、場景壓力記憶 | 簡報、銷售腳本、職場英文、考試 |

**5 大學習指標**（報告中以分數呈現）
- 輸入偏好：視覺 / 聽覺 / 文字 / 動作
- 記憶形成方式：圖像 / 聲音 / 邏輯 / 情境 / 情緒 / 社交
- 複習節奏：高頻短練 / 低頻深練 / 任務驅動 / 期限驅動
- 輸出方式：口說 / 寫作 / 教學 / 商業應用
- 阻礙模式：拖延 / 健忘 / 卡文法 / 怕開口 / 難持續

**個人化報告**包含：記憶天份名稱（主＋副類型）、學習指標分數、記憶座標、
語言學習路線、推薦策略、複習節奏、最佳應用場景、錯誤修正方法、
阻礙破解法、30 天養成計畫。

## 使用方式

直接用瀏覽器開啟 `index.html` 即可，無需安裝任何套件。

## 部署到 GitHub Pages

1. Repo → **Settings → Pages**
2. Source 選 **Deploy from a branch**，挑選此分支
3. 上線後網址為 `https://<user>.github.io/<repo>/memory-quiz/`

> 24 題、約 5 分鐘。沒有對錯，選最像自己自然反應的選項即可。
