
## 🎯 交友系統 MVP 規劃

---

### 一、資料庫設計（6 張新表）

#### 1. `energy_points` — 能量點數帳戶
| 欄位 | 說明 |
|------|------|
| user_id | 用戶 |
| balance | 目前餘額 |
| lifetime_earned | 累計獲得 |

#### 2. `energy_transactions` — 點數交易紀錄
| 欄位 | 說明 |
|------|------|
| user_id | 用戶 |
| amount | 數量（正=獲得，負=消費） |
| type | `practice_reward` / `daily_login` / `task_complete` / `purchase` / `shop_redeem` / `match_cost` |
| description | 說明文字 |

#### 3. `user_interests` — 兩層式興趣泡泡
| 欄位 | 說明 |
|------|------|
| user_id | 用戶 |
| target_language | 正在學的語言 |
| learning_goals | 學習目標（陣列：考試準備、旅行、工作等） |
| interest_topics | 興趣話題（陣列：音樂、電影、科技等） |

#### 4. `daily_matches` — 每日配對紀錄
| 欄位 | 說明 |
|------|------|
| user_id | 用戶 |
| matched_user_id | 配對對象 |
| compatibility_score | 契合度 0-100 |
| match_date | 配對日期 |
| is_free | 是否免費（第一次免費） |
| status | `pending` / `accepted` / `rejected` |

#### 5. `friendships` — 語伴關係 & 親密度
| 欄位 | 說明 |
|------|------|
| user_a_id / user_b_id | 雙方用戶 |
| intimacy_score | 親密度分數 |
| title | 關係稱號（陌生人→點頭之交→語伴→摯友→靈魂語伴） |
| chat_bg_color | 聊天室背景色（親密度達標自動解鎖） |
| message_count | 互動訊息數 |

#### 6. `shop_items` — 配配幣商城商品
| 欄位 | 說明 |
|------|------|
| name / description | 商品名稱與描述 |
| price | 所需能量點數 |
| type | `avatar_frame` / `chat_bg` / `article_style` / `physical` |
| duration_days | 時效天數（null=永久） |
| stock | 庫存（null=無限） |
| image_url | 商品圖片 |

---

### 二、核心頁面

1. **交友大廳頁** `/friends`
   - 每日一抽卡片動畫（顯示對方頭像、語言、興趣泡泡、契合度%）
   - 接受/跳過按鈕
   - 第二次抽需支付能量點數（顯示餘額）

2. **語伴列表頁** `/friends/list`
   - 已配對的語伴卡片
   - 顯示親密度進度條、目前稱號
   - 進入語伴聊天室

3. **個人興趣設定** （在 Profile 頁新增區塊）
   - 選擇目標語言
   - 選擇學習目標泡泡（多選）
   - 選擇興趣話題泡泡（多選）

4. **配配幣商城頁** `/shop`
   - 商品列表（頭框、背景、實體商品）
   - 顯示能量餘額
   - 兌換確認彈窗

5. **能量點數頁** `/energy`
   - 餘額概覽
   - 獲取/消費紀錄
   - 每日任務列表（簽到、完成對話等）

---

### 三、核心邏輯

**能量點數獲取規則（MVP）：**
- 每日登入簽到：+10 點
- 完成一次對話練習：+20 點
- 完成每日任務：+15 點
- 付費購買：待接 Stripe

**每日一抽匹配算法：**
- 第一層：相同目標語言 → 基礎分
- 第二層：學習目標 + 興趣話題重疊度 → 契合度分數
- 每日免費一次，第二次起扣 30 點

**親密度 & 稱號系統：**
| 親密度 | 稱號 | 解鎖 |
|--------|------|------|
| 0-20 | 陌生人 | — |
| 21-50 | 點頭之交 | — |
| 51-100 | 語伴 | 聊天室變淡藍 |
| 101-200 | 摯友 | 聊天室變暖橘 |
| 200+ | 靈魂語伴 | 聊天室變漸層紫金 |

每傳送一則訊息 +1 親密度，每完成一次共同練習 +5。
